# T138：心理健康平等定理 (Mental Health Parity Theorem)  

**定理陈述**: 心理健康与身体健康具有同等重要性，应获得相同的社会关注和资源投入  

## 形式化表述  
```
Mental_Health_Value = Physical_Health_Value  
Stigma_Elimination + Resource_Parity = Mental_Health_Justice  
```

## 严格证明  

**步骤1**: 心身统一原理  
心理和身体健康密切关联，共同构成整体健康  

**步骤2**: 功能等效性  
心理障碍在功能损害方面与身体疾病等同  

**步骤3**: 污名的不合理性  
对心理疾病的社会污名缺乏科学和道德依据  

**步骤4**: 治疗效果证据  
心理治疗的有效性与身体治疗相当  

∴ 心理健康平等是全面健康观的必然要求 □
