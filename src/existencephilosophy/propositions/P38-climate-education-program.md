# P38：气候教育计划命题 (Climate Education Program Proposition)  

**理论基础**: T125 + T127 + C38  

**命题陈述**: 实施全民气候教育计划，培养气候正义意识和适应减缓能力  

## 教育内容  

### 科学知识  
```  
气候变化科学原理  
影响评估方法  
预测模型理解  
```  

### 正义原则  
```  
历史责任认知  
能力差异理解  
公平分担原则  
```  

### 行动技能  
```  
个人减排行为  
适应策略选择  
集体行动参与  
```  

## 实施方式  

学校课程、社区教育、媒体宣传、在线学习相结合的多元化教育体系。  

∴ 气候教育是建设低碳社会的文化基础 □
