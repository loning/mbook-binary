## 4. 规范化与等价

### 4.1 规范化Norm

```
Norm: FS → FS,  Norm² = Norm
```

- τ归约为约定括号正规形（如左结合）并记录逆变换的事件切片
- σ/b归约为最短词（对称群最短交换词/Garside正形）

- Norm不丢信息：所有可逆操作通过TGL⁺事件回放可路径还原

### 4.2 两层等价关系

**值等价（忽略拓扑）**：
```
FS₁ ≡_{val} FS₂ ⟺ ⟦FS₁⟧ = ⟦FS₂⟧
```

**折叠同构（拓扑等价）**：
```
FS₁ ≅_{fold} FS₂ ⟺ Norm(FS₁) = Norm(FS₂)
```

运行时可在值层做缓存/归并；审计与研究时使用折叠层，不约掉拓扑差异。

### 6.1 基本过程

| **规范群** *(Gauge Group)*            | 张量空间的局部自同构群                                                                                       | $G(x):\mathcal H_x\to\mathcal H_x$                                | 局域基变换对物理态不变的群                           |
| **规范场** *(Gauge Field)*            | 规范群的联络形式                                                                                          | $A_\mu\in\mathfrak g$                                             | 描述局域自由度变化的几何联络                          |
| **量子涨落** *(Quantum Fluctuation)*   | 张量模式的随机相位扰动                                                                                       | $\delta\phi\sim\mathcal N(0,\sigma^2)$                            | 有限维与不完全投影引起的扰动                          |
| **虚粒子** *(Virtual Particle)*       | 中间态张量模式                                                                                           | $\ket{\psi_{\mathrm{virt}}}$                                      | 非外态的中间模式                                |
| **耦合常数** *(Coupling Constant)*     | 张力传递算子的幅度因子                                                                                       | $g_{ab}$                                                          | 自由度交互强度                                 |
| **普朗克常数** $\hbar$                  | 张量生成的尺度单位                                                                                         | $\hbar=E\cdot T$                                                  | 能量-时间与自由度数目的换算单位                        |
| **光速** $c$                         | 张量相位的传播极限速度                                                                                       | $v_{\mathrm{phase}}\le c$                                         | 几何子空间中相位传播的上限                           |
| **引力常数** $G$                       | 几何张量与能量张量的耦合系数                                                                                    | $R_{\mu\nu}-\tfrac12g_{\mu\nu}R=\frac{8\pi G}{c^4}T_{\mu\nu}$     | 曲率与能量密度的比例关系                            |
| **精细结构常数** $\alpha$                | U(1) 子空间的归一化系数                                                                                    | $\alpha=\frac{e^2}{4\pi\epsilon_0\hbar c}$                        | 电磁子空间的归一化张力值                            |
| **信息维度** *(Information Dimension)* | 统一场态下可区分信息方向数                                                                                     | $\dim_{\mathrm{info}}\mathcal H\le \dim_{\mathrm{alg}}\mathcal H$ | 例如强作用：SU(3) 李代数维数 8，经等价类压缩得信息维 4        |
